SkillSync AI

AI-powered resume and skill gap analyzer built with FastAPI, React, Groq LLMs, Render, and Vercel.

Live Demo
Frontend
https://skillsync-ai-zeta.vercel.app
Backend API
https://skillsync-ai-vsq1.onrender.com
API Docs
https://skillsync-ai-vsq1.onrender.com/docs

FEATURES
-Resume PDF upload
-AI-powered resume analysis
-Skill gap detection
-Personalized learning roadmap
-Job description matching
-FastAPI backend
-React + Vite frontend
-Groq LLM integration
-Fully deployed cloud architecture
